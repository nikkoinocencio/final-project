TR
TR is used to create rows within a table.
TH
TH is used to add headers to tables which provide a clearer and more organized appearance.
TD
TD is used to add content to tables.
THEAD
Thead used to group header content in an HTML table.
TFOOT
TFOOT is used to group footer content in an HTML table.
CAPTION
CAPTION is used to title a table.
COLSPAN vs ROWSPAN
COLSPAN combines columns horizontally while ROWSPAN combines rows vertically.
CELLSPACING vs CELLPADDING
CELLSPACING defines the space between adjacent cells while CELLPADDING defines the space between a table's border and content.
